O Restoid lhe dá controle total sobre os backups de seus aplicativos através de uma interface limpa, moderna e simples. Utilizando o motor de código aberto "restic", foi criado para usuários que exigem backups robustos, criptografados e com deduplicação.

O acesso root é estritamente necessário para ler e gravar em diretórios de dados privados de aplicativos.

Recursos:
* Baseado no Restic: Aproveita a velocidade, a segurança e a eficiência do restic para backups locais, deduplicados e criptografados.
* Backup Seletivo: Escolha exatamente quais aplicativos instalados pelo usuário você deseja salvar.
* Controle Granular: Selecione detalhadamente o que incluir para cada aplicativo: arquivos APK, dados do usuário, dados protegidos do dispositivo, externo/OBB ou arquivos de mídia.
* Gerenciamento Flexível de Repositórios: Crie e gerencie repositórios através dos backends nativos do restic: diretório local, SFTP, servidor REST e Amazon S3/MinIO.
* Gerenciamento de Snapshots: Navegue facilmente pelos snapshots de backup históricos, veja detalhes do que foi salvo e remova dados antigos para economizar espaço.
* Restauração Cirúrgica: Restaure aplicativos inteiros ou apenas partes específicas (como restaurar apenas os dados do aplicativo sem sobrescrever o APK).
* Agendamentos Automatizados: Configure backups recorrentes com intervalos e condições de disparo personalizáveis (bateria, carregamento e status da rede).
* Proteção contra Downgrade: Evita que você restaure acidentalmente uma versão mais antiga de um aplicativo sobre uma mais recente.
