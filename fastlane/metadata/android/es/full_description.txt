Restoid le ofrece un control total sobre las copias de seguridad de sus aplicaciones a través de una interfaz de usuario limpia, moderna y sencilla. Impulsado por el motor de código abierto "restic", está diseñado para usuarios que exigen copias de seguridad robustas, cifradas y deduplicadas.

Se requiere acceso root para leer y escribir en los directorios de datos privados de las aplicaciones.

Características:
* Impulsado por Restic: Aprovecha la velocidad, seguridad y eficiencia de restic para copias de seguridad locales, deduplicadas y cifradas.
* Copia de seguridad selectiva: Elija exactamente qué aplicaciones instaladas por el usuario desea guardar.
* Control granular: Seleccione minuciosamente qué incluir para cada aplicación: archivos APK, datos de usuario, datos protegidos por el dispositivo, datos externos/OBB o archivos multimedia.
* Gestión flexible de repositorios: Cree y gestione repositorios en los backends nativos de restic: directorio local, SFTP, servidor REST y Amazon S3/MinIO.
* Gestión de instantáneas: Explore fácilmente las instantáneas de copias de seguridad históricas, vea los detalles de lo que se guardó y elimine datos antiguos para ahorrar espacio.
* Restauración quirúrgica: Restaure aplicaciones completas o solo partes específicas (como restaurar solo los datos de la aplicación sin sobrescribir el APK).
* Programaciones automatizadas: Configure copias de seguridad recurrentes con intervalos y condiciones de activación personalizables (batería, carga y estado de la red).
* Protección contra degradación: Evita que restaure accidentalmente una versión antigua de una aplicación sobre una más reciente.
