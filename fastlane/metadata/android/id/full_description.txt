Restoid memberi Anda kendali penuh atas cadangan aplikasi melalui antarmuka pengguna yang bersih, modern, dan sederhana. Ditenagai oleh mesin sumber terbuka "restic", aplikasi ini dibuat untuk pengguna yang menginginkan cadangan yang tangguh, terenkripsi, dan terdeduplikasi.

Akses root sangat diperlukan untuk membaca dan menulis direktori data aplikasi pribadi.

Fitur:
* Didukung Restic: Memanfaatkan kecepatan, keamanan, dan efisiensi restic untuk pencadangan lokal, terdeduplikasi, dan terenkripsi.
* Pencadangan Selektif: Pilih aplikasi yang diinstal pengguna mana yang ingin Anda simpan.
* Kendali Granular: Pilih secara mendetail apa yang disertakan untuk setiap aplikasi: file APK, data pengguna, data yang dilindungi perangkat, eksternal/OBB, atau file media.
* Manajemen Repositori Fleksibel: Buat dan kelola repositori di seluruh backend restic asli: Direktori lokal, SFTP, server REST, dan Amazon S3/MinIO.
* Manajemen Snapshot: Telusuri snapshot cadangan historis dengan mudah, lihat detail apa yang dicadangkan, dan hapus data lama untuk menghemat ruang.
* Pemulihan Bedah: Pulihkan aplikasi sepenuhnya, atau hanya bagian tertentu (seperti memulihkan data aplikasi saja tanpa menimpa APK).
* Jadwal Otomatis: Atur pencadangan rutin dengan interval dan kondisi pemicu yang dapat disesuaikan (baterai, pengisian daya, dan status jaringan).
* Perlindungan Penurunan Versi: Mencegah Anda memulihkan versi aplikasi lama secara tidak sengaja di atas versi yang lebih baru.
