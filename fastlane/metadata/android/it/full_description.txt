Restoid ti offre il controllo completo sui backup delle tue app attraverso un'interfaccia utente pulita, moderna e semplice. Basato sul motore open source "restic", è progettato per gli utenti che richiedono backup robusti, crittografati e deduplicati.

L'accesso root è strettamente necessario per leggere e scrivere nelle directory dei dati privati delle applicazioni.

Funzionalità:
* Basato su Restic: Sfrutta la velocità, la sicurezza e l'efficienza di restic per backup locali, deduplicati e crittografati.
* Backup selettivo: Scegli esattamente quali applicazioni installate dall'utente desideri salvare.
* Controllo granulare: Seleziona in modo granulare cosa includere per ogni app: file APK, dati utente, dati protetti dal dispositivo, file esterni/OBB o file multimediali.
* Gestione flessibile dei repository: Crea e gestisci repository tramite i backend nativi di restic: directory locale, SFTP, server REST e Amazon S3/MinIO.
* Gestione degli snapshot: Sfoglia facilmente gli snapshot di backup storici, visualizza i dettagli di ciò che è stato salvato ed elimina i vecchi dati per risparmiare spazio.
* Ripristino chirurgico: Ripristina intere app completamente, oppure solo parti specifiche (come il ripristino dei soli dati dell'app senza sovrascrivere l'APK).
* Pianificazioni automatizzate: Imposta backup ricorrenti con intervalli e condizioni di attivazione personalizzabili (batteria, ricarica e stato della rete).
* Protezione dal downgrade: Impedisce di ripristinare accidentalmente una versione di un'app più vecchia su una più recente.
