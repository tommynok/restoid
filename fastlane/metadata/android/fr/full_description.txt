Restoid vous donne un contrôle total sur vos sauvegardes d'applications via une interface utilisateur propre, moderne et simple. Propulsé par le moteur open-source « restic », il est conçu pour les utilisateurs exigeant des sauvegardes robustes, chiffrées et dédupliquées.

L'accès root est strictement requis pour lire et écrire dans les répertoires de données privées des applications.

Fonctionnalités :
* Propulsé par Restic : Exploite la vitesse, la sécurité et l'efficacité de restic pour des sauvegardes locales, dédupliquées et chiffrées.
* Sauvegarde sélective : Choisissez exactement les applications installées par l'utilisateur que vous souhaitez sauvegarder.
* Contrôle granulaire : Sélectionnez précisément ce qu'il faut inclure pour chaque application : fichiers APK, données utilisateur, données protégées de l'appareil, fichiers externes/OBB ou fichiers multimédias.
* Gestion flexible des dépôts : Créez et gérez des dépôts sur les backends natifs de restic : répertoire local, SFTP, serveur REST et Amazon S3/MinIO.
* Gestion des instantanés : Parcourez facilement les instantanés de sauvegarde historiques, consultez les détails des éléments sauvegardés et supprimez les anciennes données pour libérer de l'espace.
* Restauration chirurgicale : Restaurez entièrement les applications ou seulement des parties spécifiques (par exemple, restaurer uniquement les données d'application sans écraser l'APK).
* Planification automatisée : Configurez des sauvegardes récurrentes avec des intervalles et des conditions de déclenchement personnalisables (batterie, charge et état du réseau).
* Protection contre les retours aux versions antérieures : Vous empêche de restaurer accidentellement une ancienne version d'une application sur une version plus récente.
